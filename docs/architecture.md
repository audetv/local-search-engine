# Архитектура Local Search Engine

## Обзор

Приложение состоит из двух основных компонентов:
1. **Core library** (`lse_core`) — поисковый движок без UI
2. **Desktop UI** — Qt 6 интерфейс

## Core Library

### Слои

1. **Tokenizer** — токенизация текста, разбиение на слова
2. **Stemmer** — морфологический анализ (стемминг Snowball для русского)
3. **Index** — инвертированный индекс
   - In-memory index (Phase 1)
   - On-disk index (Phase 2)
4. **Search** — выполнение запросов, BM25 ранжирование
5. **Parser** — извлечение текста из форматов книг
6. **Chunker** — разбиение на параграфы
7. **Metadata** — хранение метаданных в SQLite
8. **Remote** — HTTP-клиент для Manticore Search

### Поток данных при поиске

```
Query → Tokenizer → Stemmer → Index::search() → Ranker → Results
```

### Поток данных при индексации

```
Book → Parser → Chunker → Tokenizer → Stemmer → Index::addDocument()
```

## Решения

### C++20
Выбран для использования современных возможностей (span, expected, ranges, concepts).

### vcpkg
Пакетный менеджер для воспроизводимых сборок на всех платформах.

### Статическая линковка
Цель — один бинарник без внешних зависимостей.

### mmap для индекса
Для быстрого доступа к большим файлам индекса без полной загрузки в память.

### SQLite для метаданных
Встроенная БД, не требует сервера, отлично подходит для десктопного приложения.
