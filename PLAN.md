# План разработки: Локальный поисковый движок (Local Search Engine)

## Статус: черновик · версия 0.1 · 2026-07-15

---

## 1. Концепция продукта

**Что:** Десктопное приложение для полнотекстового поиска по русскоязычным книгам.  
**Как:** Локальный индекс своей библиотеки + опциональный поиск через HTTP к серверу Manticore Search (корпус 176k+ книг).  
**Для кого:** Исследователи, читатели, владельцы личных библиотек.  
**Отличие:** Один установщик, не требует Docker/MySQL/специальных знаний. Мгновенная установка и работа.

## 4. Принципы разработки

1. **Итеративность:** Каждый этап завершается работающим прототипом.
2. **Метрики успеха:** Измеряем время поиска, размер индекса, точность на каждом этапе.
3. **Zero-overhead:** Не платим за то, что не используем.
4. **RAII и безопасность:** Никаких утечек, санитайзеры в CI.
5. **Один бинарник:** Финальная цель — самодостаточный исполняемый файл.
6. **Документирование решений:** ARCHITECTURE.md с обоснованием каждого крупного выбора.
